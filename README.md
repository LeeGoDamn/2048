# 🎮 2048

经典的 2048 数字合并游戏，使用 TypeScript + Vite + Tailwind CSS 构建。

## 🎮 游戏地址

**https://LeeGoDamn.github.io/2048/**

## ✨ 特性

- 🎯 **经典玩法** - 合并相同数字，达到 2048
- ⌨️ **键盘控制** - 方向键或 WASD
- 📱 **触摸滑动** - 移动端完美支持
- 📊 **分数统计** - 当前分数 + 最佳记录
- 🎨 **炫酷动画** - 方块出现/合并动画
- 💾 **本地存储** - 自动保存最佳记录

## 🎯 游戏规则

1. 每次移动，所有方块会向指定方向滑动
2. 相同数字的方块碰撞时会合并
3. 每次移动后会随机出现一个新方块（2 或 4）
4. 当无法移动时游戏结束
5. 合并出 2048 即获胜！

## 🕹️ 操作说明

| 操作 | 键盘 | 移动端 |
|------|------|--------|
| 向上 | ↑ / W | 向上滑动 |
| 向下 | ↓ / S | 向下滑动 |
| 向左 | ← / A | 向左滑动 |
| 向右 | → / D | 向右滑动 |
| 新游戏 | 点击按钮 | 点击按钮 |

## 🚀 本地开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build
```

## 💡 技巧提示

- 把大数字放在角落
- 尽量保持一行/一列有序排列
- 不要随意向上移动（除非必要）
- 提前规划下一步
